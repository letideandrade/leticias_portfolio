    Check out my web design and mobile application projects!

    Projects in this section were developed for different purposes and for several different companies, but most of them were created using HTML for page structure, CSS for style information and JavaScript as main Programming Language.

  • Fomapp Vegan App Project

    Fomapp App Project - Part I: The Fomapp Vegan App project is a new mobile application dedicated to vegan lifestyle enthusiasts. The app aims to provide users with resources, recipes, restaurant recommendations based on their location, and community support, emphasizing sustainability and ethical living. The app features a distinctive logo and color palette that resonate with the target audience. It is available for both Android and iOS users.

  • Fomapp Vegan App Project

    Fomapp App Project - Part II: The Fomapp Vegan App is a new mobile application for vegan lifestyle enthusiasts. The second part of this project involved the complete development of all the app's pages to provide users with relevant resources, recipes, restaurant recommendations based on location, and vegan community support, all while emphasizing sustainability and ethical living. The app is available for both Android and iOS users, with UX and UI concepts widely applied throughout the project.

  • Finans Website

    Finans Project - Part I: This project involved developing a website for Finans, a financial management company. The website is fully responsive and was created with HTML, CSS, JavaScript, and Bootstrap. This is the initial page of the website. The full project is presented in Part II, which is also included in this section of the portfolio.

  • Finans Website - Part II: Full Project

    Finans Project - Part II: This is the full project for Finans, a financial management company. The website aims to help clients gain a better understanding of where to invest their money and how to manage their savings. It is a responsive webiste, allowing these tasks to be performed on mobile devices as well. It was developed with HTML, CSS, JavaScript, and Bootstrap.

  • Noticias Cidade Website

    Noticias Cidade Project: This is a local breaking news Brazilian website, developed as a platform to deliver real-time updates and information pertinent to the largest cities in Brazil. The main menu allows residents to choose from different topics and easily find information, making the platform user-friendly and intuitive.

  • Tecblog Website

    TecBlog Project: Tecblog is a simple and intuitive platform that delivers the latest global technology news with clarity and readability. Its fully responsive design ensures that updates are seamlessly accessible across all devices. The project was developed with HTML, CSS and JavaScript.

  • Chale Hotel - Nature Hotel & Luxury Resort

    Chale Hotel Project: A Nature Hotel & Luxury Resort website represents the business of a family that has been providing a dreamy and relaxing experience to many visitors for five generations. This platform was specially created for hotel guests and day-pass spa visitors, allowing them to easily book reservations, purchase tickets, check seasonal events sponsored by Chale Hotel, and browse through all amenities and fun family activities offered. Concepts of UI and UX were actively applied during the development of this project.

  • Color Harmony Generator

    Color Harmony Generator Project: In the dynamic world of web design, color selection is not just about aesthetics—it's a critical component that influences user experience and brand identity. Understanding this, the Color Harmony Generator was created: a state-of-the-art background generator website designed specifically for web designers who want to match and harmonize colors effortlessly, with features for cohesive design, real-time previews, and background simulations.

  • Brazilian National Museum Website

    Museu Nacional Project: The Brazilian National Museum website receives a lot of traffic every day, primarily because it is also used as the platform where visitors purchase tickets and check current events, such art exhibitions happening at the museum. The platform features an intuitive user interface, allowing visitors to navigate easily and enjoy a smooth browsing experience.

  • Anna's Model Portfolio

    Model Portfolio Project: This website was developed to showcase the portfolio of models from a modeling agency in Brazil. The agency needed a webpage where clients could view and select models that best suited their needs. This example illustrates the webpage's structure, which allows for easy updates and modifications to accommodate each model's details and professional photos. It was designed using HTML, CSS, and JavaScript. The final result was highly effective, enabling the agency to present their models to potential clients efficiently.

  • Flor da Idade Website Project

    Flor da Idade - Website Project: This web design project is dedicated to the senior community. The buttons are enlarged and well-highlighted to help older adults find topics of interest, thereby improving their user experience while browsing. The website, Flor da Idade, which translates to "Ageless Flower," seeks to celebrate and empower individuals in their golden years through insightful content and engaging narratives. It aims to attract a broader readership and strengthen its position as a leading voice in senior living and lifestyle publications.
